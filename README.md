-------------------------------------------------------------------------------
📋 Instructions exécution
------------------------------------------------------------------------------- 

📋 UBUNTU LINUX

Grant execution permissions to the script:

    
    chmod +x setup_full_ubuntu.sh


Run the script:

    ./setup_full_ubuntu.sh
       
Check the log file:

    cat /mnt/data/testuser/setup_log.txt


------------------------------------------------------------------------------

    chmod +x setup_full.sh
